# 📌 Project Overview

This project focuses on analyzing sales performance data using Python for data preparation and Power BI for interactive dashboard development. The objective was to identify revenue trends, profit performance, discount impact, regional sales distribution, and product-level insights to support data-driven business decisions.

# 🎯 Objectives
1) Analyze overall business performance using key sales metrics.
2) Identify top-performing products and categories.
3) Evaluate sales distribution across regions and states.
4) Understand revenue contribution by different business segments.
5) Create an interactive dashboard for business stakeholders.


# 📂 Project Workflow
1. Data Preparation (Python) : 
Imported and explored the dataset.
Handled missing values and data inconsistencies.
Performed data cleaning and preprocessing.
Validated data quality before visualization.

3. Dashboard Development (Power BI) : 
Created KPI cards for key business metrics.
Built interactive visualizations and filters.
Designed an intuitive dashboard layout for easy analysis.

# 🔍 Key Insights
1) Identified the most profitable products contributing to business growth.
2) Analyzed regional and state-wise sales performance.
3) Compared sales across different customer segments and product categories.
4) Evaluated the impact of discounts on overall business performance.
5) Highlighted categories generating the highest revenue.

# 📬 Conclusion
This dashboard provides a comprehensive view of sales, revenue, profit, and discount performance, enabling stakeholders to monitor business health, identify growth opportunities, and make informed decisions through interactive visual analytics.
